Nossos cursos

Venha conhecer nossos cursos e entender como levamos a inovação para a vida das crianças!
Desenvolvimento de Games 2D

Faixa Etária: 5 até 7 anos

Tenho interesse!

O curso de desenvolvimento de games em duas dimensões permite que a criança crie seus próprios jogos no computador através de uma linguagem de programação simples e didática, além de roteirizar e criar o design do game. Esses conceitos que a criança aprenderá servirão para desenvolver seu pensamento crítico e habilidades de resolução de problemas. Elas aprendem como criar e se expressar com o computador, não apenas para interagir com ele.

Desenvolvimento de Games 3D

Faixa Etária: 7 até 10 anos

Tenho interesse!

O curso de desenvolvimento de games em três dimensões permite que a criança crie seus próprios jogos no computador através de uma linguagem de programação visual, simples e didática, além de roteirizar e criar o design do game, trabalhando com a noção de profundidade.

Desenvolvimento de Aplicativos

Faixa Etária: a partir de 8 anos

Tenho interesse!

O curso de desenvolvimento de aplicativos tem como objetivo o ensino da criação de aplicativos para o sistema operacional Android através de uma interface gráfica onde os alunos aprendem a desenvolver o layout do app e a programação do mesmo através da linguagem blockly, uma adaptação da linguagem de programação para o ensino a crianças.

Robótica com drones

Faixa Etária: a partir de 9 anos

Tenho interesse!

Este curso tem como objetivo o ensino robótica, aplicando os conceitos a dinâmicas com drones. Trabalhando durante o curso a eletrônica, com a realização dos circuitos internos dos drones, montagem dos componentes, programação dos componentes eletrônicos e do vôo, além da pilotagem de drones.
